## Project Ideas  
- Redesign a **popular app UI** (Spotify, Instagram, YouTube)  
- Create a **Portfolio Website UI** in Figma/Adobe XD  
- Design a **Mobile App Prototype** (e.g., Food Delivery)  
- Build a **Design System / Style Guide** for a website  
- Work on a **Case Study Project** (problem-solving design approach)
